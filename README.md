# ایران رادیو

برنامه دسکتاپ پخش زنده شبکه‌های رادیویی ایران.

این برنامه نسخه غیررسمی [ایران صدا][1] است و از API های ایران صدا استفاده می‌کند.


## دانلود برنامه

* [Windows x64][3]

تمامی نسخه‌های ایران رادیو در [صفحه دانلود برنامه][3] در دسترس هستند.
